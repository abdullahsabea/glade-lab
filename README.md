# GLADE — Coming Soon Landing Page

Public teaser site for the GLADE project, deployed via GitHub Pages.

**Live:** https://abdullahsabea.github.io/glade-lab/

Built with React 19 + Vite + Tailwind CSS v4 + Framer Motion.

## Local dev

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` automatically deploy via GitHub Actions → GitHub Pages.
